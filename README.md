# QuickText Voice Pro - PWA

Application web progressive de transcription vocale, traduction et synthese IA.

## Fonctionnalites

- 🎤 Dictee vocale en temps reel
- 🌐 Traduction automatique multilingue
- 🤖 Synthese et correction par IA (OpenRouter)
- 🔊 Lecture audio du texte
- 📄 Extraction de texte PDF
- 💾 Export audio MP3 et texte TXT
- 📱 Installation PWA sur mobile et desktop
- 🔒 Cle API chiffree localement

## Installation

```bash
# Test local
npx serve .

# Deploiement Netlify
netlify deploy --prod

# Deploiement Vercel
